Question-2: What is the difference between innerHTML, innerText, and textContent?

1. innerText is basically used to get what is visible to us in a website, ignores whatever the backend codes.
2. innerHTML is consists of of the HTML content which has the tags and elements
3. textContent is used for to get/set all raw texts including hidden without formatting

Question-3: What is event delegation in the DOM?

It is an event listener technique applied to the parent element so that whenever any child is clicked, the event follows back to the parent

Question-4: What is event bubbling in the DOM?

if a certain area is clicked then all the parent areas which stores this certain area will be clicked. this process of event moved towards the parent is called event bubbling.
